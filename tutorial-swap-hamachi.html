Creating a SWAP partition for your Alcatel One Touch Fire

This was made on Lubuntu using GParted, could change depending the used tool.

1.- Get a SD card memory of class 10 or 8 as minimum. If you don't have a class 10 or 8 then don't do it.

2.- Open GParted or your prefered tool to manage partitions and select your device.

3.- Delete the actual partition, after select the option to create a new one.

4.- Set "New Size" as 512 MB and "File System" as linux-swap.

5.- Create a new partition with the rest of the available space and set the "File System" as fat32.

6.- Apply all the changes. This will erase all your data, so make a backup of your files.

7.- This is how should seems your SD card.

8.- Now you can put it in your phone.

9.- Now you must execute the correct file to enable swap in the system, the file is "swap-*.sh" where * is your OS type.
Windows: doubcle click on "swap-windows.bat"
Linux: Open your terminal and type "./swap-linux.sh"
Mac: Open your terminal and type "./swap-mac.sh"
All this files and the other necessary files are with the images to flash your phone or directly here!.
