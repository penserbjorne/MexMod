Setting Udev Rule for your device

This works on Ubuntu, could change depending your GNU/Linux distribution

1.- Download the file from here (direct download) or here (Mega).

2.- Open the terminal and type gksudo nautilus

3.- Type your password in the dialog box.

4.- Navigate to /etc/udev/rules.d

5.- Unzip the .zip and copy the file from the unzipped folder.

6.- Paste the file in the folder /etc/udev/rules.d.

7.- Close nautilus or press Ctrl + C in the terminal.

8.- Type in the terminal sudo chmod a+r /etc/udev/rules.d/51-android.rules to make the file readable.

9.- Type in the terminal sudo service udev restart to restart the service and apply the changes.
