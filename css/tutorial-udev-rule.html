Setting Udev Rule for your device

This works on Ubuntu, could change depending your linux distribution

1.- Download the file from here (support us!) or here (direct download).

2.- Open the terminal and type gksudo nautilus

3.- Type your password in the dialog box.

4.- Navigate to /etc/udev/rules.d

5.- Unzip the .zip and copy the file from the unzipped folder.

5.- Paste the file in the folder /etc/udev/rules.d.

6.- Close nautilus and the terminal.
